# UNI Tax Farm

## Tech Stack

- React
- TailwindCSS
- ShadcnUI
- TypeScript
- TanStack Query + Router
- Vite
- Wagmi

## Prerequisites

- bun 1.1.16

## Development

```bash
bun install
bun dev
```
